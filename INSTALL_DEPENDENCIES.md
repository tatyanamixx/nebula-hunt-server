# Установка зависимостей для Nebulahunt Server

## Обновленные зависимости

### Удалено

-   `qrcode` - модуль с критической уязвимостью, не используется в коде

### Добавлено

-   `axios` - для HTTP запросов к Google OAuth API

## Установка

```bash
cd nebulahunt-server
npm install
```

## Проверка безопасности

```bash
npm audit
```

## Зависимости для Google OAuth

Основные зависимости для работы Google OAuth + 2FA:

-   `axios` - HTTP клиент для запросов к Google API
-   `speakeasy` - генерация и валидация 2FA кодов
-   `jsonwebtoken` - JWT токены для сессий
-   `sequelize` - ORM для работы с базой данных

## Проверка установки

После установки зависимостей проверьте, что сервер запускается:

```bash
npm run dev
```

Если возникают ошибки с модулями, выполните:

```bash
npm install --force
```

## Безопасность

-   Все зависимости проверены на уязвимости
-   Удалены неиспользуемые модули с уязвимостями
-   Используются актуальные версии пакетов
